Principais Formatações

Negrito / Destaque

Podemos negritar uma frase Com a tag B (Não semântica)

E podemos Com a tag strong(semântica)

Italico / Ênfase

Podemos por um termo em Italico com a tag I (Não semântica)

Podemos dar Ênfase num termo usando a tag em (semântica)

Marcador de texto

Podemos marcar um texto com a tag Mark

Podemos mudar a marca de um texto com a tag Mark

Texto Grande e pequeno

Vamos criar um texto grande e um texto pequeno nesse paragráfo

Texto deletado

Podemos meter um texto deletado, que serve para indicar que o texto deve ser lido, mas não considerado

Texto inserido

Podemos marcar um texto como inserido, de maneira a indicar que o texto foi inserido depois com uma certa ênfase

Podemos sublinhar também um texto com a tag u

Texto sobrescrito

Textos sobreescritos são potências, ex: x45+5

Texto subscrito

Texto subscrito são indices, ex: H2O2
